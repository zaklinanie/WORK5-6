Что такое JavaScript?

JavaScript - это язык программирования, который используется для создания интерактивных веб-сайтов и приложений.
С помощью JavaScript можно добавлять динамику на статические веб-страницы и обрабатывать пользовательский ввод.

Какие возможности есть у JavaScript?

  • Cоздание интерактивных элементов на странице
  • Обработка пользовательских действий (например, нажатие на кнопку)
  • Валидация форм и проверка введенных данных
  • Анимация элементов
  • Использование AJAX для загрузки данных асинхронно без перезагрузки страницы
  • Работа с графикой
  • Работа с базами данных

Когда не стоит использовать JavaScript?

Когда на сайте нет сложной логики и взаимодействия с пользователем. Например, в случае статического сайта-визитки или лендинга.

Демонстрация возможностей JavaScript

  • Приветствие

    Представьтесь, пожалуйста, чтобы мы могли обращаться к Вам по имени на протяжении всей работы со страницей.

  • Площадь треугольника

    Введите основание и высоту треугольника для поиска его площади.

  • Сравнение строк

    Введите 2 различные строки для сравнения количества символов. Если количество совпало, будет выведено true, в противном случае false.

  • Поиск минимума и максимума в массиве

    Введите массив из 5 элементов для поиска минимального и максимального значения массива.

  • Таймер

    Функции таймера: начать отсчет, остановить отсчет, посмотреть сколько времени прошло между стартом и финишем.

  • Тестирование

    Пройдите тест из 10 вопросов по JavaScript. После прохождения будет доступен результат и ответы на вопросы с указанием "верно/неверно".

  • Заставка

    При нажатии на кнопку появляется заставка на весь экран с красивым фоном, текущей датой и именем пользователя.
